What is GLauncher?

GLauncher is a lightweight helper for Windows that focuses on one thing: opening the right set of applications together so you can get into your flow faster.

Instead of hunting through the Start menu or taskbar, you define groups like "Sim racing", "Dev environment" or "Morning office" and launch them in a single click.

Highlights
  • Group any desktop apps together, including custom executables.
  • Use a curated app list or search and add your own programs.
  • Control launch order, add command-line flags and force admin per app.
  • Start with Windows, minimize to tray and choose light or dark theme.

Key features

GLauncher focuses on control and convenience without trying to be an entire desktop environment.

Group-based launching

Create as many groups as you like and launch them from the tray or main window.

Launch order

Control the order your apps start in, with delays where you need them.

Developer flags & admin

Attach command-line flags and choose to run specific apps as administrator.

Smart app detection

Start from a curated list of common programs, then search and add your own executables.

Start with Windows

Optionally run GLauncher on login so your groups are always one click away.

Light/dark mode & tray

Pick the theme you like and keep GLauncher tucked away in the Windows tray.
